# Customer_data_behaviour_analysis
Data analytics project in which the raw data has been collected and done the ETL process in the python, using MySQL got insights about the data and finally created a interactive dashboard using PowerBI
Data Analytics Project: Customer Insights AnalysisOverviewThis project analyzes customer data to derive actionable insights, focusing on trends, patterns, and key metrics. The goal is to support data-driven decision-making through exploratory data analysis (EDA), SQL querying, and interactive dashboards.

Dataset- Name: Customer Transactions Dataset
- Description: Contains records of customer transactions, and purchase history.

Tools- Python: Pandas. (for EDA and data cleaning)
- MySQL: SQL querying and data extraction
- Power BI: Interactive dashboard creation
- Jupyter Notebook: Code documentation and execution

Steps1. Data Loading & EDA: Loaded dataset into Python; performed exploratory analysis to identify patterns and missing values.
2. Data Cleaning: Handled missing data, outliers, and formatted columns for consistency.
3. SQL Queries: Ran queries in MySQL to extract key metrics (e.g., top customers, sales trends).
4. Dashboard Creation: Built a Power BI dashboard visualizing insights (e.g., revenue trends, customer segments).
5. Reporting: Documented findings and insights in a structured report.

Dashboard- Highlights:
    - Customer demographics and segment-wise engagement

Results- Identified key customer segments driving revenue.
- Uncovered seasonal trends in sales data.
- Provided actionable insights for targeted marketing strategies.
